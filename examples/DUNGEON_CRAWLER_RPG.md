## ğŸ¯ How to Play

### Starting the Game
```bash
cd /home/ruckus/code/DeskApp/examples
python dungeon_crawler_rpg.py
```

### Main Menu Controls
- **ENTER**: Start new game
- **L**: Load saved game  
- **Q**: Quit game

### Character Creation
1. Enter your character name
2. Choose class (1-Fighter, 2-Wizard, 3-Rogue)
3. Read the story introduction

### Town Controls
- **S**: Visit the Shop
- **R**: Rest at the Inn (restores HP/MP)
- **D**: Enter the Dungeon
- **I**: View Inventory and character status
- **Q**: Save game and return to main menu

### Shop Controls
- **1-9**: Buy item by number
- **B**: Back to town

### Dungeon Controls
- **ENTER**: Enter the current floor
- **B**: Return to town

### Combat Controls
- **A**: Attack with weapon
- **M**: Cast magic spell (if available)
- **I**: Use item from inventory
- **R**: Run from battle (33% chance to fail)

### Character Progression
- Gain experience by defeating monsters
- Level up for increased HP, MP, and combat effectiveness
- Buy better equipment to survive deeper floors
- Learn new spells (class-dependent)
